C++ installation
================

1. **Dependencies**: Ensure you have CMake (>=3.10) and a C++14 compatible compiler.
2. **Clone the repository**:

   .. code-block:: bash

      git clone https://github.com/codac-team/codac.git
      cd codac

3. **Build the library**:

   .. code-block:: bash
   
      mkdir build
      cd build
      cmake ..
      make
